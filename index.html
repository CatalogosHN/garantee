Reglamento interno de garantía

Este reglamento explica de forma sencilla cómo se revisan las solicitudes de garantía en ventas realizadas por Facebook, WhatsApp u otros medios similares.

Lectura importante
La garantía no es aprobación automática. Todo producto debe ser revisado para confirmar si la falla corresponde a defecto de fábrica y no a daño físico, golpe, caída, humedad, rayones, manipulación, mal uso o situaciones similares.

Resumen principal

  • La garantía aplica únicamente por fallas reales atribuibles a defecto de fábrica.
  • Todos los productos pasan por revisión antes de entregarse o despacharse.
  • Si se solicita garantía, el producto será revisado nuevamente.
  • Recibir el producto para revisión no significa que la garantía ya fue aprobada.
  • No aplica por rayones, golpes, caídas, humedad, piezas quebradas, manipulación, desgaste normal, mal uso o daño estético.

Puedes aceptar con este resumen o seguir leyendo el reglamento completo.

Reglamento completo de garantía

Reglamento

1. Finalidad del reglamento

Este reglamento tiene como finalidad explicar de manera clara el proceso de revisión de garantía para productos vendidos por medios informales o digitales, como Facebook, Messenger, WhatsApp u otros canales similares.

2. Cuándo puede aplicar la garantía

La garantía puede aplicar cuando el producto presenta una falla de funcionamiento atribuible a defecto de fábrica, siempre que no existan señales de daño físico, mal uso, manipulación, humedad, caída o alteración del producto.

3. Revisión antes de entrega

Los productos son revisados antes de ser entregados o despachados. Esta revisión puede incluir verificación visual, funcionamiento básico, accesorios, estado físico y condición general del producto.

4. Revisión al solicitar garantía

Cuando el cliente solicita garantía, el producto debe pasar por una nueva revisión. La recepción del producto no significa aprobación inmediata. La garantía será aprobada únicamente si la revisión confirma que la falla corresponde a defecto de fábrica.

5. Casos donde no aplica garantía

  • Rayones, golpes, abolladuras, quebraduras, fisuras, caídas o daño visible.
  • Humedad, líquidos, corrosión, sulfatación o exposición al agua.
  • Manipulación, apertura, reparación, modificación o instalación realizada por terceros.
  • Uso incorrecto, sobrecarga eléctrica, accesorios inadecuados o instalación incorrecta.
  • Desgaste normal, suciedad, manchas, deterioro estético o daño cosmético.
  • Daños ocasionados por aplicaciones, configuraciones, software no autorizado, virus, formateos o cambios realizados por el cliente.

6. Resultado de la revisión

Después de revisar el producto, se informará al cliente si la garantía procede o no procede. Si procede, se podrá ofrecer reparación, cambio, solución técnica o gestión correspondiente, según el tipo de producto y disponibilidad.

7. Derechos del consumidor

Este reglamento busca ordenar el proceso de garantía de forma clara y transparente. Ninguna condición aquí indicada pretende eliminar derechos que correspondan al consumidor conforme a la normativa aplicable.

Al aceptar, el cliente confirma que leyó y entendió el reglamento de garantía. También comprende que la garantía no cubre daños físicos, rayones, caídas, humedad, manipulación, mal uso ni situaciones similares.
